# Bank Management System

## Features
- Account management (create, view, deposit)
- Fund transfer with multithreading
- MySQL integration
- REST API with Spring Boot
- Docker deployment

## Usage
1. `docker-compose up --build`
2. API: `http://localhost:8080/api/accounts`
